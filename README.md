不需要服务器，不需要复杂的部署过程，个人类型的小程序就可以玩。

#### 使用说明

1.  去腾讯位置服务(https://lbs.qq.com)
申请获取美食商家的key
2.  将pages/play/play.js中第45行左右的key替换
3.  申请属于自己的美团、饿了么推广链接，替换原来的链接
